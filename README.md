## Clone

```bash
git clone git@github.com:exactly-finance/protocol.git
```

## Install Locally

```bash
npm install
```

## Run Tests

```bash
npm test
```

## Run Coverage

```bash
npm run coverage
```

## Gas Reports

```bash
export REPORT_GAS=1 && npm test
```

## Deploy

```bash
npm run deploy:kovan
```
